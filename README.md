## 🚀 Getting Started
### Requirements
- PHP 7.4 or higher
- A web server (Apache / Nginx) or PHP built-in server

### Running locally
```bash
# Clone the repository
git clone https://github.com/AOgit/php_oop.git
cd php_oop

# Run a specific lesson using the PHP built-in server
php -S localhost:8000 -t 01/
```
Then open [http://localhost:8000](http://localhost:8000) in your browser, or run files directly from the command line:
```bash
php 01/index.php
```

## 🧠 What You'll Learn
- Core OOP concepts: classes, objects, properties, and methods
- Encapsulation, inheritance, polymorphism, and abstraction
- Abstract classes and interfaces, including type control
- PHP magic methods (`__construct`, `__destruct`, `__get`, `__set`, `__toString`, etc.)
- Namespaces, PSR-4 autoloading, and Composer
- Traits and late static binding
- Common design patterns in PHP
